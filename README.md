# Infinite Productions

Immersive interactive digital identity card for Infinite Productions.

## Run

```bash
npm.cmd install
npm.cmd run dev -- --port 5173
```

## Asset Hooks

Place provided brand files here:

- Logo: `public/assets/logo.png`
- High Cruiser font: `public/assets/fonts/HIGHCRUISER.woff2` or `public/assets/fonts/HIGH CRUISER.ttf`
- A Day Without Sun Text font: `public/assets/fonts/ADayWithoutSunText.woff2` or `public/assets/fonts/A DAY WITHOUT SUN TEXT.ttf`

The experience uses CSS-driven haze, grain, glow, and particles, so it remains atmospheric even before smoke reference files are added.
